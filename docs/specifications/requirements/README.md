## 📋 概要

Works Logue（ワークスログ）は技術者・エンジニア向けの記事投稿・知識共有プラットフォームです。

### 対象ユーザー

- **一般ユーザー**: 技術記事を読みたい開発者・エンジニア
- **投稿者**: 技術知識・経験を共有したい開発者・エンジニア
- **管理者**: プラットフォーム運営・管理を行う運営チーム

### プロジェクト範囲

- Web アプリケーション（デスクトップ・モバイル対応）
- 記事投稿・編集・閲覧機能
- ユーザー認証・プロフィール管理
- リアクション・コメント機能
- 検索・タグ機能
- 管理機能
